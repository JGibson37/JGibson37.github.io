Virtual Pets Amok

Virtual Pets Amok is a java program where you assume the role of a pet shelter worker. It allows you to play with, feed, give water, adopt out, and accept pets into the shelter. Some of the main concepts used in the program include Object Oriented Programming, Test Driven Development, ArrayLists,inheritance, and polymorphism.
